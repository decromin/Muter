Muter

Muter is a chrome extension that allows individual users to be muted, or "modified" within Campfire chat sessions.

What it is

This is a a Chrome browser extension. It provides control over noisy users or bots within Campfire chat sessions.

Muter adds a button to the Campfire room interface, and upon clicking it a menu is presented.

Clicking on an option in the popup menu will result in that user being muted from chat. Using the "User says" option with modify the specified users chat messages. This only effects the local user!

Why

Some bots (or users) need to be given a time-out, so this tool allows for that, without having to kick or remove the offender.
